# This is a new doc for Node JS Mongo DB and GraphQL

### Building and running your application

1. When you're ready, start your application by running:
`docker compose up --build`.

### Aleternatively you can follow below steps to build and run the container separately!

1. If you want to build image with the latest code: 
`docker compose build`.

2. And then you can run the below command to start the docker container:
`docker compose up` .
